# Сделайте кнопки на странице, при нажатии на которые появляются сообщения. Первая кнопка с надписью «Работает?», при нажатии на неё появляется сообщение «Работает. Не трогай!». Вторая кнопка с надписью «Хм…», при нажатии на неё появляются по очереди три цитаты (на ваш выбор), которые хранятся в программе в массиве.
